# Project Status

NordicWebHub is currently a demo-ready full-stack MVP.

## Completed

- Public landing and pricing pages
- Admin and Customer authentication
- HttpOnly cookie authentication
- CSRF protection
- Login lockout
- Role-based authorization
- One Customer equals one Company data isolation
- Admin dashboard
- Customer dashboard
- Company management
- Service package management
- Project requests
- Projects
- Support tickets and replies
- Service Orders / Payment Status demo
- Maintenance logs
- SEO reports
- Hosting status and manual website health check
- AI Service Assistant
- Frontend design system and polished layouts
- Public documentation and screenshot checklist

## In Progress

- Final screenshots for README and portfolio documentation
- Deployment preparation
- GitHub publishing review

## Known Limitations

- Service Orders do not process real payments.
- AI Service Assistant uses rule-based recommendations.
- Optional external AI SEO generation requires a separately configured API key.
- Production deployment is not included in the repository.
- Email notifications are not implemented.
- File uploads are not implemented.
- Team/member accounts are not implemented.

## Planned Improvements

- Stripe or another payment provider
- Optional OpenAI or Azure OpenAI integration
- Email notifications
- File uploads for projects and tickets
- Team/member accounts
- Scheduled website health checks
- More automated integration tests
- Production deployment pipeline
