# Future Improvements

NordicWebHub is a demo-ready MVP. The following improvements would make it stronger for production or a larger portfolio version.

## Product Improvements

- Customer onboarding flow
- File uploads for project assets and support tickets
- Notification center
- Email notifications
- Timeline view for projects
- More detailed project milestones
- Customer comments on maintenance logs and SEO reports
- Advanced package filtering and comparison
- Service order invoice export

## Admin Improvements

- Admin user management
- Staff assignment to projects and tickets
- Internal notes
- SLA tracking for support tickets
- Audit log for important status changes
- Export reports as PDF

## Technical Improvements

- More integration tests for authorization rules
- More frontend component tests
- Production deployment profile
- Centralized structured logging
- Background job for scheduled website checks
- Real email service integration
- Real payment provider integration
- Better API versioning strategy
- Rate limiting for auth endpoints

## Security Improvements

- Production secret storage
- Strict production CORS configuration
- HTTPS-only deployment
- Security headers
- More complete audit logging
- Stronger password and account recovery workflow

## AI Improvements

- Optional integration with Azure OpenAI or OpenAI
- Admin-controlled prompt templates
- Saved recommendation templates
- Better output validation
- Clear cost controls and request limits

## UX Improvements

- More polished mobile layouts
- Improved empty-state illustrations
- Better table filtering and sorting
- Global search
- Dark mode
- More detailed dashboard insights
